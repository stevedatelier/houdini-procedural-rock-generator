Houdini Detailed Rocks is a procedural rock generator designed for creating a wide range of rock formations directly inside Houdini.

The setup includes customizable controls for shape, scale, surface breakup, and detail generation, allowing artists to quickly create variations for environments, product visualization, motion graphics, and close-up renders.

Included:

* Houdini project file
* Procedural rock generation setup
* Textures and maps
* Example materials and render setup

Important: This project requires the cache files and textures included in the external download package.
