# Houdini Procedural Rock Generator

> A fully procedural rock generation system built in Houdini — designed for creating a wide range of rock formations with fine-grained artistic control, fast variation, and production-ready output.

## Overview

Houdini Detailed Rocks is a procedural rock generator with customizable controls for shape, scale, surface breakup, and detail generation. Built for artists who need quick, believable rock variations across environments, product visualization, motion graphics, and close-up renders — without hand-sculpting every piece.

## What's Included

| Component | Description |
|---|---|
| Houdini Project File | Complete `.hip` file, ready to open |
| Procedural Rock Setup | Fully parametric generation network |
| Textures & Maps | All surface maps included |
| Example Materials | Pre-configured render-ready materials |
| Render Setup | Lighting and camera example scene |

## Getting Started

### Requirements

- **Houdini 19.5** or later
- The external cache files and textures (distributed separately — see below)

### Installation

**1. Clone or download this repository**

```bash
git clone https://github.com/steveghst/houdini-procedural-rock-generator.git
```

**2. Download the cache files and textures**

Required assets are distributed separately due to file size:

> **[Download Cache Files and Textures — Google Drive](https://drive.google.com/drive/folders/1eY6CiogXlD80Tr3clifM1XHqeFFyAzEB?usp=drive_link)**

Place the downloaded files into the project root so Houdini can resolve all paths on load.

**3. Open in Houdini**

Open the `.hip` file. All procedural controls are exposed at the top-level network.
